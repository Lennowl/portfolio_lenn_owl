Bowl Of Dice

Novembre 2023 - En cours

Bowl Of Dice est un projet de bot Discord que j'ai commencé en Novembre 2023. Le but de ce bot discord est de permettre aux joueurs du jeu de rôle Donjons et Dragons d'avoir accès à plusieurs fonctionnalités du jeu, tel que le lancé de dé ou la création de personnages depuis un serveur Discord.

Les différents aspects que j'ai abordés lors de la réalisation de ce projet sont :

- Création d'un projet informatique : J'ai commencé ce projet de zéro avec pour seule ambition, créer un bot discord pour les joueurs de donjons et dragons! J'ai donc du définir ce que je voulais implémenter dans ce bot.
- Implémenter une base de données relationnelle : L'une des fonctionnalités du bot étant la création de personnage, il a fallu créer entièrement une base de données pour le permettre.
- Soigner l'écriture du code : Un bot discord fonctionne avec énormément de commandes, le code qui permet de les gérer peut donc rapidement s'accumuler. Il faut ainsi penser à avoir de bonnes pratiques pour maintenir un code sain.
