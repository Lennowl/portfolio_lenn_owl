Lenn_owl

Aventuriers du rails

Mai 2023 - Juin 2023

Le projet des aventuriers du rails a été réalisé en duo lors de mon deuxième semestre de BUT. Nous devions programmer le célèbre jeu de société en redefinissant les principales fonctionnalités de celui-ci dans le code. Nous avons également réalisé une interface graphique du jeu en JavaFX pour permettre aux joueurs d'avoir une bonne interaction avec le jeu.

Les différents aspects que j'ai abordés lors de la réalisation de ce projet sont :

- Utilisation de JavaFX : Afin de créer l'interface utilisateur permettant de jouer au jeu, nous avons utilisé la bibliothèque JavaFX pour proposer une bonne expérience visuelle.
- Algorithmie avancée : Pour retrouver le chemin le plus cours possible pour un joueur, il fallait utiliser des méthodes de parcours de graphes avancées tel que l'algorithme de Dijkstra.
- Implémentation de test : Afin de tester le bon fonctionnement de l'application, nous avons implémenter une série de tests. Ces tests avaient pour but de vérifier le bon fonctionnement de nos fonctions et de réveler de potentiels problèmes.
