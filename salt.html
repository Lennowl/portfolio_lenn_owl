SAlt

Septembre 2023 - Janvier 2024

SAlt est un site web réalisé lors de mon troisième semestre de BUT avec 3 autres personnes. Le but du site est d'aider les étudiants et les enseignants a mieux gérer l'édition des conventions de stages et des alternances. Le site permet également aux entreprises de déposer des offres à destination des étudiants.

Les différents aspects que j'ai abordés lors de la réalisation de ce projet sont :

- Utilisation de la méthode agile SCRUM : Pour la gestion de projet nous avons adoptés la méthodes SCRUM. Notre projet était donc réparti en plusieurs sprints. Au début de chaque sprint nous faisions une réunion pour définir les tâches à faire. Nous faisions des réunions tous les jours pour s'informer de l'avancée de nos tâches respectives.
- Création d'interface utilisateurs : Un point clé de notre projet était l'accesibilité de notre application. Nous avons donc fait notre maximum pour rendre l'interface du site simple et compréhensible.
- Bonne pratique de code : Pour ce projet, nous avons utilisé principalement le langage PHP. Afin d'avoir une structure de code correct nous avons donc implémenté le modèle MVC qui découpe le code en trois partie distinctes. Ce modèle permet de rendre un code de meilleur qualité car mieux organisé.
